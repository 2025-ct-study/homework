### BOJ 14501 퇴사
- dp
- dp[ i ] : i번째 날까지 얻을 수 있는 최대 수익
  - dp[ i ]는 i번째 날과 i-1번째 날 수익 중 최댓값
    - dp[ i ] = max(dp[ i ], dp[ i-1 ])
  - i번째 날 상담 시 상담이 끝나는 날 `j = i + T[i]`의 최대 수익을 갱신
    - dp[ j ] = max(dp[ j ], dp[ i ] + P[ i ])
- dp[ N+1 ]값이 퇴사일까지 얻을 수 있는 최대 수익
