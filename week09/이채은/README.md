### BOJ 2529 부등호
- 백트랙킹: 0~9 사이의 숫자를 순서대로 추가
- 숫자 추가 시, 아래 조건 확인
  - 방문한 적 없는 수 인지
  - 부등호 조건에 맞는 수 인지
- 끝(N+1)까지 도달하면, 최대/최소 값 갱신

### BOJ 17298 오큰수
- 이중 탐색 시 시간초과 발생 
  - 스택을 사용하여 개선 
  - 스택: 오큰수를 찾아야하는 수(인덱스)를 담는 용도, 처음엔 비어있음
- 배열을 차례대로 탐색 (for)
  - 스택이 비어있지 않고, 현재 탐색하는 수가 스택에 있는 수 보다 크면 (while)
    - 현재 탐색하는 수는 스택에 있는 수의 오큰수
    - 오큰수를 찾았으니 스택에서 pop
  - 현재 탐색하는 수 스택에 추가
- 배열 전체 탐색 후, 스택에 남아있는 수는 오큰수를 찾지 못한 수 이므로 -1 처리
