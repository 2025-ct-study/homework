### BOJ 3190 뱀
- 시뮬레이션
  - 덱 사용: 덱의 First를 뱀의 머리로, 덱의 Last를 뱀의 꼬리로 생각
  - 큐 사용: 방향 전환 정보 (시간, 회전 방향)
- 방향 전환이 필요한 시간이면 방향 전환
- 방향에 맞게 다음 위치로
- 다음 위치가 벽이거나 자기 몸이면, break
- 그게 아니면 다음 위치로 머리 이동
  - 사과가 있으면 사과 없애기
  - 사과가 없으면 꼬리 없애기
